How the predictions work

Running every method on every benchmark would cost thousands of GPU-hours, so we don't. The cells we did run are real measurements (✓/ℹ/◇); the rest are predicted — the ~values, shown with a 95% range and never counted as verified.

The prediction is a plain statistical model. From the cells we have measured it learns three things, then adds them up for any empty cell:

  • method skill — how strong each model generally is,
  • dataset difficulty — how hard each benchmark generally is,
  • protocol effect — how much each eval recipe shifts the numbers.

It's the same idea as guessing a student's grade on an exam they skipped, from their other grades and how hard each exam is. The 95% range is wide when there's little nearby evidence and narrows as we measure more — so a wide range flags a cell we're unsure about. Two honesty rules keep it grounded: a method is only predicted for tasks it can actually do (depth-only models never appear under the pose tabs), and predictions are clamped to each metric's valid range (e.g. δ₁ ≤ 1).

We also let the predictions choose what to run next: the widest-range (most uncertain) cell is the most informative, so we run it and check the result. If the measurement lands inside the predicted range, the model was right; when it doesn't, we've found behaviour the model couldn't have guessed (e.g. DA3's unexpectedly-high DIODE error). That run-the-most-uncertain-cell loop is how the beyond-the-papers ◇ cells were filled — each predicted first, then measured, with both numbers in the hover.

Metrics are grouped depth · pose · trajectory. The pose and trajectory tabs include the companion numbers each paper row also reports (RRA@15 / RTA@15, RPE-trans / RPE-rot°), audited the same way; the “47 verified results” counts the primary metric per reproduction, not these companions. Tolerance is pinned per-cell: ±5% for nearly all cells, ±10% for the three video/trajectory-pose cells (sub-cm ATEs amplify small fp16-vs-bf16 drift), and ±2% for DA3 on NYU. Only one cell exceeds 5% — DAGE Sintel at +7.3%, inside its ±10% band. Cells outside their pinned tolerance don't count as verified and live in DISCREPANCIES.md. Every value is audited against the source PDF (table + column + row) — AUDIT.md. Some baselines are cited from a later paper's comparison table (e.g. DA-V2 / DUSt3R rows measured in MoGe / MASt3R) — the hover names the exact source.
