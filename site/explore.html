Benchmark scores under one protocol at a time.

Scores are only comparable under the same eval recipe, so you pick a protocol (a paper's eval table) and the matrix shows just that one — every column is then a fair comparison, and colour ranks weak→best within it. Each cell is the score (the metric value): ✓ our run reproduces that paper's number, ℹ it differs (hover for the deviation + why), ◇ has no paper target. ~values are predictions — a Bayesian fit over (method × dataset × protocol × metric) that shares method-skill across metrics (a method seen only in δ₁ still gets an AbsRel score), filling each unrun cell a method can actually produce — depth-only methods are absent from the pose tabs and vice-versa. The best in each column is bold. Only the 47 ✓ scores are paper-verified. Switch metric with the tabs, protocol with the pills; hover any cell for the paper number, the deviation, and the citation.

Metrics are grouped depth · pose · trajectory. The pose and trajectory tabs include the companion numbers each paper row also reports (RRA@15 / RTA@15, RPE-trans / RPE-rot°), audited the same way; the “47 verified results” counts the primary metric per reproduction, not these companions. Tolerance is pinned per-cell: ±5% for nearly all cells, ±10% for the three video/trajectory-pose cells (sub-cm ATEs amplify small fp16-vs-bf16 drift), and ±2% for DA3 on NYU. Only one cell exceeds 5% — DAGE Sintel at +7.3%, inside its ±10% band. Cells outside their pinned tolerance don't count as verified and live in DISCREPANCIES.md. Every value is audited against the source PDF (table + column + row) — AUDIT.md. Some baselines are cited from a later paper's comparison table (e.g. DA-V2 / DUSt3R rows measured in MoGe / MASt3R) — the hover names the exact source.
